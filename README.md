## 🚀 What is ValidateFast?

**ValidateFast** is a SaaS validation kit designed for founders who want to **stop building in the dark**. Instead of spending months perfecting a product nobody wants, ValidateFast gives you everything you need to prove (or disprove) your idea in just 72 hours.

It ships as a **ready-to-deploy Next.js landing page** with a mock dashboard, so you can show potential early adopters exactly what they'll be paying for — before you write a single line of backend code.

## 📦 Getting Started

### Prerequisites

- Node.js `v18+`
- npm or yarn

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/your-username/validatefast.git
cd validatefast

# 2. Install dependencies
npm install

# 3. Start the development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the app.

### Available Scripts

| Command | Description |
|---|---|
| `npm run dev` | Start local development server |
| `npm run build` | Build the production bundle |
| `npm run start` | Start the production server |

## 💡 How to Use It

1. **Clone & customize** the landing page copy to match your SaaS idea
2. **Deploy to [Vercel](https://vercel.com)** in one click (zero config required for Next.js)
3. **Drive traffic** to your landing page (Twitter, Reddit, Product Hunt, etc.)
4. **Collect emails** and measure interest over 72 hours
5. **Analyze results** — if people sign up, build it. If not, pivot fast.
